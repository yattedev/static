# static
## Installation  
```$ npm install ```  
or  
```$ yarn ```  
## Getting Started  
starting dev-server  
```$ npm start ```  
or  
```$ yarn start  ```  
  
starting mock-server  
```$ yarn json-server db/db.json --port [DB_PORT]```  
## Testing 
```$ npm test ```  
or  
```$ yarn test  ```
